Install the Desktop App

This project ships a Tauri desktop app. To run it locally or build installers:

  • Install Rust and Tauri prerequisites as described in DESKTOP_BUILD.md.
  • Run the dev desktop app: npm run tauri:dev.
  • Build installers: npm run tauri:build (artifacts in src-tauri/target).

If you distribute via a store or releases, link users to your official downloads.
